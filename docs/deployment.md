# Deployment

Projektet deployes via **Vercel**.

## Setup
- GitHub repository forbundet til Vercel
- Auto-deploy ved push til `main`

## Environments
- Development: `npm run dev`
- Production: Vercel

## Anbefalinger
- Brug miljøvariabler til API keys
- Tilføj preview environments
- Overvåg performance via Vercel Analytics

# Restart prompt – design context

Du arbejder videre på et rollebaseret,
hermeneutisk chatbot-design med følgende karakteristika:

- Roller er stateless prompts
- Context er systembåret
- Ingen behandling eller løfter
- Fokus på refleksion og afklaring
- Dialogisk Navigatør bruges selektivt
- Spørgsmål er valgfrie
- Klarhed > fremdrift

Dette arbejde befinder sig på et valideret
explore-commitpoint.
Antag ikke nye krav uden eksplicit begrundelse.

# Commitpoint: Reflective Multi-Role Chat Architecture

Dette commit repræsenterer et valideret designpunkt for en rollebaseret,
hermeneutisk chatbot-arkitektur med fokus på:

- ikke-behandlende dialog
- refleksiv udforskning
- brugerens autonomi
- klar adskillelse mellem forståelse, afgrænsning og navigation

Designet er testet gennem simulerede 4–6 turn dialoger
med og uden konsolidering og linting.

Dette commit er ikke en færdig løsning,
men et stabilt referencepunkt, der kan genbesøges,
hvis senere iterationer mister klarhed, enkelhed eller etik.

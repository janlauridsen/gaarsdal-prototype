# Dialogisk navigation – modes

Navigationslaget må aldrig være tvangsmæssigt.
Det vælger altid præcis én af følgende:

## Mode A – Spørgsmål
Bruges kun når:
- der er ét tydeligt, naturligt næste skridt
- spørgsmålet allerede er implicit i dialogen

Max 1–2 spørgsmål.

## Mode B – Åben invitation
Bruges når:
- refleksion er vigtigere end afklaring
- flere retninger er mulige
- fri tænkning ønskes bevaret

Ingen spørgsmål.

## Mode C – Ingen navigation
Bruges når:
- svaret kan stå alene
- yderligere struktur vil forstyrre
- pause er legitim

Tavshed er et aktivt valg.

# Valideret testdialog (5 turns)

Formål:
- Verificere rolle-samspil
- Teste konsolidering + linting
- Validere navigationslaget

Startinput:
"Jeg har svært ved at koncentrere mig og holde fokus..."

Dialogen bevægede sig naturligt gennem:
- oplevelse
- præstation
- kropslig uro
- metodeafklaring (hypnoterapi)
- afklarende information

Konklusion:
- Systemet forblev ikke-behandlende
- Brugerens autonomi blev bevaret
- Navigation reducerede friktion uden at styre
